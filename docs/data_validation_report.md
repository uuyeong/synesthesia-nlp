## 1. Eagleman Synesthesia 데이터

### 개요
- **출처:** Eagleman et al. (2007) 공감각 연구 논문 부속 데이터  
- **형태:** 알파벳 26자 각각에 대해 수천 명의 공감각자가 응답한 색상의 평균 RGB  
- **파일:** `synesthesia_grapheme_mean_rgb.csv`

### 검증 결과

| 항목 | 내용 |
|---|---|
| 데이터 쌍 수 | 26개 (A–Z) |
| 피험자 수 | 알파벳당 5,972 ~ 6,510명 평균 |
| 제공 형식 | 0~1 정규화 값 + 0~255 RGB 값 동시 제공 |
| 결측값 | 없음 |
| 포맷 | `grapheme, n_subjects, R_0_1, G_0_1, B_0_1, R_0_255, G_0_255, B_0_255` |

### 샘플

| 알파벳 | R | G | B | 색상 경향 |
|---|---|---|---|---|
| A | 196 | 81 | 51 | 붉은 계열 |
| R | 135 | 53 | 51 | 진한 붉은 계열 |
| Y | 192 | 175 | 73 | 노란 계열 |
| I | 178 | 179 | 145 | 밝은 회백 계열 |

### 단점 및 제한사항
- **26개뿐** — 단독으로는 Linear Probe 학습 데이터로 부족 (다른 데이터와 반드시 병합 필요)
- 개인차가 있는 공감각 데이터를 평균낸 값이라 노이즈 내포
- 문자(grapheme) 레벨 데이터이므로 단어(word) 레벨 모델과 입력 레벨이 다름

## 2. NRC Word-Colour Association Lexicon

### 개요
- **출처:** Saif Mohammad, NRC Canada (v0.92)  
- **취득 방법:** GitHub 미러 (`beefoo/text-analysis`) 에서 다운로드  
- **파일:** `NRC-color-lexicon.txt`, `nrc_word_rgb.csv`  
- **원본 형식:** `word--sense[TAB]Colour=X[TAB]VotesForThisColour=N[TAB]TotalVotesCast=N`

### 검증 결과

| 검사 항목 | 결과 | 비고 |
|---|---|---|
| 총 단어 수 (unique) | 14,182개 | word-sense 기준 24,206개 |
| 색상 카테고리 수 | 11개 | black, blue, brown, green, grey, orange, pink, purple, red, white, yellow |
| binary 형태 (0/1) | 아님 | 투표수(votes/total) 방식 |
| RGB 수치 존재 | 없음 | 색상명 → RGB 변환 필요 |
| 결측값 비율 | 26.6% | Colour=None 행 |
| 라이선스 | 연구용 무료 | NRC 공개 리소스 |
| RGB 변환 후 사용 가능 쌍 | **11,449개** | 색상명 → RGB 매핑 후 |

### RGB 변환 매핑 (적용 기준)

```
red=(220,20,20), orange=(255,140,0), yellow=(255,215,0),
green=(34,139,34), blue=(30,100,200), purple=(128,0,128),
pink=(255,105,180), brown=(139,69,19), white=(255,255,255),
black=(30,30,30), grey=(128,128,128)
```
- 한 단어에 여러 색상 연관 → 연관 색상들의 RGB 평균  
- Colour=None 단어 → 제외

### 변환 결과 직관성 검증

| 단어 | 변환 RGB | 색상 | 직관적? |
|---|---|---|---|
| fire | (220, 20, 20) | 빨강 | ✓ |
| ocean | (30, 100, 200) | 파랑 | ✓ |
| forest | (34, 139, 34) | 초록 | ✓ |
| night | (30, 30, 30) | 검정 | ✓ |
| snow | (255, 255, 255) | 흰색 | ✓ |
| anger | (220, 20, 20) | 빨강 | ✓ |
| love | (255, 105, 180) | 분홍 | ✓ |
| death | (30, 30, 30) | 검정 | ✓ |
| sky | (30, 100, 200) | 파랑 | ✓ |
| joy | — | 없음 | ✗ |

### Eagleman 데이터와 병합 시
- NRC 11,449개 + Eagleman 26개 = **총 11,475쌍**

### 단점 및 제한사항
- **sense-level 데이터:** `word--sense` 형태로 같은 단어가 의미별로 여러 행 존재 → 단어 추출 시 sense 정보 소실, 의미 혼합 가능성
- **Binary 아님:** 0/1이 아닌 투표 비율 방식 → 직접 binary 활용 불가, 전처리 필요
- **RGB 없음:** 색상명만 있어 별도 매핑 필요하며, 매핑 기준이 주관적
- **추상 단어 누락:** joy, happy 등 일부 감정 단어 Colour=None 처리됨
- **색상 평균의 왜곡:** red+blue 평균 → 보라색 중간값이 실제 의도와 다를 수 있음
- **공식 배포 방식:** 이메일 신청 필요 (GitHub 미러 사용)

## 3. Gutenberg Poetry Corpus

### 개요
- **출처:** Project Gutenberg 공개 도메인 시 컬렉션 (Allison Parrish 정제)  
- **취득 방법:** HuggingFace `datasets` 라이브러리 (`biglam/gutenberg-poetry-corpus`)  
- **저장 파일:** `poetry_word2vec.model`, `poetry_candidate_words.txt`

### 검증 결과

| 항목 | 내용 |
|---|---|
| 전체 줄 수 | 3,085,117줄 |
| 시집/책 수 | 1,191개 (gutenberg_id 기준) |
| 컬럼 구조 | `line`, `gutenberg_id` |
| 전체 토큰 수 | 22,653,428개 |
| Unique 단어 수 (원본) | 225,929개 |

### 정방향 용도: Word2Vec 학습 결과

| 항목 | 결과 |
|---|---|
| 학습 어휘 수 | **69,379개** |
| 벡터 차원 | 300차원 |
| 학습 설정 | window=5, min_count=5, epochs=10 |
| Seed 단어 전체 포함 여부 | **모두 포함** (R/G/B 각 10개 seed 전부) |

### 시 코퍼스 Word2Vec vs Google News Word2Vec 비교

| 단어 | 시 코퍼스 RGB | Google News RGB | L2 차이 |
|---|---|---|---|
| fire | (97, 76, 81) | (95, 80, 79) | 4.9 (작다) |
| ocean | (76, 80, 98) | (73, 81, 99) | 3.3 (작다) |
| forest | (73, 96, 85) | (75, 99, 80) | 6.2 (작다) |
| night | (80, 83, 91) | (90, 77, 86) | 12.7 (작다) |

→ **최대 L2 차이 12.7** — 두 모델 간 색상 결과가 거의 동일. 시 코퍼스만으로 충분한 품질.

### 역방향 용도: 후보 단어 집합

| 항목 | 결과 |
|---|---|
| 필터링 조건 | 영문자만 + 길이 3+ + 빈도 50회 이상 + stopwords 제거 |
| 필터링 후 | 18,145개 |
| BERT 어휘 교집합 후 | **9,953개** |
| 시적 표현 단어 포함 | **18/18 전부 포함** |

시적 표현 단어 빈도 예시:

| 단어 | 빈도 | 단어 | 빈도 |
|---|---|---|---|
| soul | 19,736회 | bright | 13,008회 |
| dark | 11,601회 | moon | 8,005회 |
| dream | 7,975회 | river | 5,740회 |
| flame | 5,612회 | silence | 5,042회 |
| grief | 5,434회 | dawn | 4,838회 |

### 단점 및 제한사항
- **고어 다수 포함:** `thy`(71,471회), `thou`(63,872회), `thee`(43,900회) 등이 최상위 빈도 — 역방향 후보 단어 집합에서 별도 필터링 필요
- **Cosine Similarity 방식의 한계:** snow, dawn 등 색상이 모호한 단어는 결과값이 중립적(~85,84,87)으로 나와 색상 구분력 낮음
- **Anchor 벡터 주관성:** seed 단어 선택에 따라 anchor 벡터가 달라지며, 결과 RGB도 변동됨
- **줄 단위 데이터:** 문장/문단이 아닌 시의 한 줄씩 저장되어 있어 문맥 창(window)이 줄 경계에서 끊김
